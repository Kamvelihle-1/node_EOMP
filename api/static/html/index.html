Router Description

HTTP Methods PATH Description
GET / or /home Home page
GET /users List of all users
GET /user/:id Get a user
POST /user User sign up/registration
POST /user User login
PATCH /user/:id Update user with new information
DELETE /user/:id Delete a user
GET /products List of all products
GET /product/:id Get/view one product
POST /product Add a product
PATCH /product/:id Update product information
DELETE /product/:id Delete a product
